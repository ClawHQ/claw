The operator cockpit for OpenClaw agents that run while you sleep.

  <img src="https://claw.so/images/logo.jpeg" alt="Claw Logo" width="120" />
</p>

## What is Claw?

Claw Desktop is the native app for reviewing and managing OpenClaw agent runs. Deploy OpenClaw anywhere, chat via Slack while it works, then open Claw Desktop to review artifacts and resume runs.

### Features

- **While-You-Were-Away Sync** — Resume runs instantly, no scrolling through Slack history
- **Artifact Review** — Diffs, outputs, and verification checklists in one place
- **One-Click Resume** — Continue the same run_id across Slack and Desktop
