## Naming Patterns

### By Content (AI-Powered)

```
Rename screenshots based on what's visible in the image
```

_Uses AI vision to understand image content_

### By Date

```
Add the file creation date to all PDFs: YYYY-MM-DD-filename.pdf
```

### By Pattern

```
Rename invoice PDFs to: Vendor-InvoiceNumber-Date.pdf
```

### By Metadata

```
Rename photos using their EXIF date: 2025-01-03-IMG_4832.jpg
```

## Undo Any Rename

Made a mistake? Every rename is logged:

1. Open **History** in Ishi
2. Find the rename operation
3. Click **Undo**

The file reverts to its original name instantly.
