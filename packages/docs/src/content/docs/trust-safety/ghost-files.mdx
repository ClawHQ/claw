## How It Works

### 1. You Make a Request

```
Rename all invoices to include the vendor name and date
```

### 2. Ishi Simulates the Changes

Ishi analyzes your files and calculates what would happenâ€”but **doesn't execute anything yet**.

### 3. You See the Ghost Preview
