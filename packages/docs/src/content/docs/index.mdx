## Connect Your AI

Ishi needs an AI provider to power its intelligence. We recommend **Ishi Intent Engine** for the easiest setup‚Äîit's included with your license and automatically routes tasks to the best model.

1. Open Ishi and run `/connect`
2. Select **Ishi** and sign in at [ishi.so/auth](https://ishi.so/auth)
3. Add billing details and copy your API key
4. Paste when prompted‚Äîyou're ready to go!

Already have an API key from Anthropic, OpenAI, or Google? [Connect any provider](/docs/providers).

Ishi **never touches your files without approval**. Every proposed change shows a preview:

- **Before:** Current file/folder state
- **After:** What it will look like

This is the "Glass Box" philosophy‚Äîyou see everything before it happens.
