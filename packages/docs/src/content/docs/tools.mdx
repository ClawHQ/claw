## Enabling & Disabling

All capabilities are **enabled by default**. You can turn specific ones off if needed:

```json title="ishi.json"
{
  "tools": {
    "bash": false,
    "webfetch": false
  }
}
```

This would prevent Ishi from running commands or accessing the web.

## Task Tracking

Ishi can maintain a to-do list while working on complex tasks:

```json title="ishi.json"
{
  "tools": {
    "todowrite": true,
    "todoread": true
  }
}
```

This helps Ishi stay organized when you ask it to do multi-step operations like "organize all my Q4 invoices by vendor."
